GuardPoint – Cybersecurity Awareness Platform

GuardPoint is an interactive platform designed to improve cybersecurity awareness. It features a password strength tool built on NCSC guidelines and integrated with the Have I Been Pwned API.

The platform includes phishing simulations where users analyse emails, highlight threats, and learn to spot attacks. Safe web browsing training is gamified to make risk identification engaging and effective.

GuardPoint was built using JavaScript, Node.js, Express, MongoDB, and OpenAI APIs. The design follows GDPR compliance and aligns with ISO 27000 standards.
